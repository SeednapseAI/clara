## Usage

Firstly, set an environment variable with you OpenAI API key:

```
export OPENAI_API_KEY="XXXXXX"
```

Then, use the command:

```
$ clara chat [PATH]
```

If the path is omitted, '.' will be used.

To exit use `CTRL-C` or `CTRL-D`.
